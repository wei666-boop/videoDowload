<h2>这是一个用来下载bilibili视频的web应用</h2>

<h3>小提示:这个项目是我第一次写的，我可以毫不留情的说这个项目现在非常烂，如果你真的有什么下载需求的话，请找正规下载软件</h3>

<h3>当然，如果你非要尝试一下的话，那么下面是使用教程(其实就是一句话)</h3>

首先克隆项目

```shell
git clone https://github.com/wei666-boop/videoDowload
```

然后你的电脑上需要有go语言相关的工具链，克隆完成后，进入根目录，执行:

```shell
go run .
```

这样项目就会在你的电脑的5443端口进行监听，在浏览器中访问localhost:5443来使用

如果你觉得我的那个背景不好看的话，你可以在static/image这个目录下换上你喜欢的照片
